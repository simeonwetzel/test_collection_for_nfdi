Metadata Relevance Annotation

Help improve Earth science data discovery. You'll rate how relevant dataset records are to a given search query. This study is fully anonymous.

Before you start

How to judge relevance

For each query you'll see 5 dataset records. Read the title and abstract, then decide how well the record would satisfy someone searching for that query.

3
Highly relevant
The dataset directly addresses the query. A researcher would be satisfied finding this record.
2
Partially relevant
The dataset is related and useful, but only covers part of the query or is a broader/narrower match.
1
Marginally relevant
The dataset touches the topic but would not typically be selected as a useful result for this query.
0
Not relevant
The dataset is unrelated to the query. A researcher would not benefit from finding this record.
Worked example
Query soil moisture measurements Central Europe
Long-term soil moisture time series for Germany 2000–2020
Exact topic, exact region, long temporal coverage — directly satisfies the query.
3 — Highly
Hydrology and soil properties of the Danube basin
Central Europe, soil data, but focuses on hydraulic properties rather than moisture measurements.
2 — Partial
European land cover classification 1990–2018
Covers the region but land cover is only tangentially related to soil moisture.
1 — Marginal
Atmospheric CO₂ concentrations at Mauna Loa Observatory
Different variable, different region — no meaningful connection to the query.
0 — Not rel.
Judge based on the metadata only — title and abstract — not on what you think the dataset might contain.
Use your domain knowledge. If a record seems relevant to you as an expert, trust that judgment.
When uncertain between two levels, pick the lower one.
